### Installation

1. Clone the repository:
```bash
git clone https://github.com/Esetobore/Mflixv1.git
```

2. Navigate to the project directory:
```bash

```

3. Install dependencies:
```bash
flutter pub get
```

4. Run the app:
```bash

```
